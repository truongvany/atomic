# atomic

A modern, interactive web project built with **Next.js + React**.

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| **Next.js 15** | React framework, App Router |
| **React 19** | Component-based UI |
| **Vanilla CSS** | Full design system via CSS Variables |
| **GSAP + ScrollTrigger** | Animations & scroll-driven effects |
| **Lenis** | Smooth inertia scrolling |

---

## 📦 Project Structure

```text
atomic/
└── frontend/
    ├── app/
    │   ├── styles/          # Modular CSS partials
    │   ├── globals.css
    │   ├── layout.jsx
    │   └── page.jsx
    ├── components/          # React components
    ├── lib/
    │   └── data.js          # Static data & configs
    ├── public/              # Assets, fonts, images
    ├── jsconfig.json
    ├── next.config.mjs
    └── package.json
```

---

## ⚙️ Setup

```bash
git clone https://github.com/truongvany/atomic.git
cd atomic/frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
